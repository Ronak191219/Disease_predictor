# 🩺 Disease Prediction System

This is a simple AI-based Disease Prediction Web App built using **Machine Learning** and **Streamlit**.

## 🚀 Features

* Predicts disease based on user symptoms
* Uses TF-IDF + Naive Bayes algorithm
* Provides basic medical advice
* Simple and interactive UI

## 🛠️ Technologies Used

* Python
* Pandas
* Scikit-learn
* Streamlit
* NLP (TF-IDF Vectorization)

## ⚙️ How It Works

1. User enters symptoms (e.g., fever cough headache)
2. Text is converted into numerical form using TF-IDF
3. Machine Learning model predicts disease
4. System shows prediction + advice
